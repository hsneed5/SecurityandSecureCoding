

This program demonstrates both symmetric and asymmetric encryption using Python.

1. Symmetric Encryption:
   - Uses AES-based encryption.
   - A single key is generated and used for both encryption and decryption.
   - The message is encrypted and then decrypted using the same key.

2. Asymmetric Encryption:
   - Uses RSA encryption.
   - A public/private key pair is generated.
   - The public key encrypts the message.
   - The private key decrypts the message.

Inputs:
- A plaintext message: "Hello, this is a secret message!"

Outputs:
- Encrypted message (for both methods)
- Decrypted message (matching original)
- Generated keys (symmetric key and RSA key pair)

Purpose:
This demonstrates the difference between symmetric and asymmetric encryption methods.
